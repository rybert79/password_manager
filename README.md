# 🔐 Password Manager CLI

Simple password manager built with Python using Fernet encryption.

## ✨ Features
- Add password
- View password
- Edit password
- Delete password
- Encryption with cryptography

## 🚀 How to use
1. Run the script:
   ```bash
   python cryp.py
   ```
2. Do not lose your key
   password_manager ignored for security
